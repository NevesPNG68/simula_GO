# GO Coffee — Simulador Gerencial

Aplicação web para simulação financeira e operacional da GO Coffee.

## Recursos

- Dashboard executivo
- Tema claro e escuro
- Layout responsivo
- Simulação financeira
- Ativação, desativação e cortes parciais de despesas
- Simulação de funcionários
- Comparação de horários
- Histórico mensal
- Projeção anual
- Melhor cenário destacado automaticamente
- Exportação para PDF e Excel
- Leitura automática de `dados_go_coffee.xlsx`

## GitHub Pages

Publique a partir da branch `main`, pasta `/root`.

O endereço esperado será:

`https://nevespng68.github.io/simula_GO/`

## Atualização dos dados

Substitua o arquivo `dados_go_coffee.xlsx` mantendo o mesmo nome. A aplicação procura a aba `Matriz` e recalcula os indicadores automaticamente.
